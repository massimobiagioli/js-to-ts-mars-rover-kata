# Mars Rover Kata: Js to Ts 

## Project Setup

* Copy and customize **.env.example** into **.env**
* Copy and customize **docker/mongodb/init-mongo.example.js** into **docker/mongodb/init-mongo.js**

## Npm Scripts

Start Server
```bash
$ npm run start
```

Test
```bash
$ npm run test
```

Start Docker (MongoDB)
```bash
$ npm run docker:up
```


Stop Docker (MongoDB)
```bash
$ npm run docker:down
```
